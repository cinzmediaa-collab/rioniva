Scope of Work: 11 Core Features

Every module below defines precisely what will be built. This is a comprehensive, enterprise-grade e-commerce upgrade.

Feature 1

State-Managed Cart

Full CRUD cart functionality synced dynamically to user accounts in Firestore. Includes a slide-in drawer, quantity updates, and persistent state across sessions.

Feature 2

Customer Profiles

Secure email/password and Google One-Tap sign in. Dedicated profile dashboard for customers to edit details and view their historical orders securely.

Feature 3

Wishlist Engine

Allows users to save products across sessions. Includes a dedicated wishlist page with 1-click functionality to move items directly into the checkout cart.

Feature 4

Colour Variants Logic

Granular control allowing admin to add dynamic hex colours per product, each supporting separate pricing, stock statuses, and Cloudinary image swaps.

Feature 5

Razorpay Gateway

Secure server-side API integration handling UPI, Cards, and Net Banking. Includes backend signature verification to prevent spoofed transactions.

Feature 6

Relational Orders

Complex schema storing items, colour variants, totals, and payment refs in Firestore. Admin manages statuses (Processing/Shipped) seamlessly.

Feature 7

SMTP Automations

Programmatic server-side triggers utilizing Resend to instantly dispatch a beautifully formatted HTML order confirmation to the customer upon payment.

Feature 8

GST Invoicing

Automated PDF generation compliant with Indian GST requirements. Calculates CGST/SGST/IGST, inserts HSN codes, and stores files in Firebase Storage.

Feature 9

Admin Operations

Central operational hub to manage all orders, alter dispatch statuses, edit live products/variants, and download PDF invoices securely.

Feature 10

Firestore Security

Strict ruleset enforcing data privacy. Customers can only read their own documents, and environment variables are strictly locked within Vercel.

Feature 11

QA, Testing & Vercel Deployment

Intense end-to-end checkout flow testing across mobile and desktop. Includes final configuration of domains, switching Razorpay from test to live mode, real ₹1 transactions, and pushing the React codebase to Vercel's global edge network.
